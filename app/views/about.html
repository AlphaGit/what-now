What does this do?

This is yet another to-do list application, but it allows you to see how the tasks depend between themselves, allowing you to make the right decision on what to work on next.

Features so far

  • Entering tasks
  • Setting tasks to depend on one or more others
  • See tasks graphed with their dependencies
  • Wrapping long names for the tasks
  • Selecting tasks in the graph
  • Selecting tasks in the table
  • Editing tasks

Why?

This application was built as an experiment to help myself try out some new technologies I had not layed my hand on yet. The ones that I already knew where AngularJS and JavaScript, while my knowledge of Karma (Testacular), Node, Express, and many others where flaky. It was also a nice experiment to try out D3.JS and build something interesting.

The following technologies where used in building this project:

  • NodeJS
  • ExpressJS
  • Grunt
  • Karma
  • Jasmine
  • AngularJS
  • d3js
  • sass-bootstrap
